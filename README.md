# BLE Emergency Button with ID Tag

## Overview

The BLE Emergency Button is a compact wireless emergency alert device designed for safety, tracking, and rapid notification applications. The device uses Bluetooth Low Energy (BLE) communication to transmit emergency signals along with a unique ID tag for identification purposes.

## Features

* Bluetooth Low Energy (BLE) communication
* Unique ID tag identification
* Compact 30 mm round button design
* Low power consumption
* Long battery life up to 3 years
* Real-time emergency alert transmission
* Lightweight and portable
* Suitable for wearable and industrial applications

## Applications

* Personal safety systems
* Elderly emergency assistance
* Industrial worker safety
* Hospital and healthcare monitoring
* Smart building automation
* Security and access systems

## Hardware Specifications

* Communication: BLE (Bluetooth Low Energy)
* Button Size: 30 mm round
* Power Source: Coin cell battery
* Battery Lifetime: Up to 3 years
* Low-power embedded controller
* LED status indication (optional)

## System Operation

1. User presses the emergency button.
2. BLE signal is transmitted instantly.
3. Emergency message includes the unique ID tag.
4. Receiver or gateway identifies the device.
5. Alert notification is generated in the monitoring system.
